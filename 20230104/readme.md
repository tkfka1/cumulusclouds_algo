## 푼 방식

갈수 있는 방향의 좌표에서 불이 난 곳이 있으면 while 문을 멈추고 count 를 출력,

또한 x*y//2+x+y 개수 만큼 반복시에 없다면 -1출력(더 많이 줄일 수 있을듯)

중간에 갈 방향이 없어져도 -1 출력
