## 푼 방식

딕셔너리에서 한줄씩 불러와서 그 전단계 딕셔너리의 index,index-1 값중에 큰값을 다시 딕셔너리에 넣는것을 반복
맨마지막줄에서 최댓값 찾아 출력

해설 : DFS ex) D[N][M] = arr[N][M] + max(D[N-1][M-1], D[N-1][M])
